# Simple Mermaid Diagram Viewer

This project is a Mermaid diagram viewer that allows users to load, render, zoom, and download Mermaid diagrams from local files.
It is built using React, TypeScript, and Mermaid.js.

## Features

- Load and render Mermaid diagrams from local files
- Zoom in and out of diagrams
- Download diagrams as SVG files

## Getting Started

1. Clone the repository to your local machine and run it
